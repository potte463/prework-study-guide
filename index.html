CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • *: apply to all elements
  • width: 100%; : how wide elements are on page
  • height: 120px; : pixel height of element
  • background-color: changes background of element
  • color:
  • text-align: alignment of text
  • ul{}: format unordered list
  • p: format paragraph
  • card: breates a box or card for the content. elements include width, margin, border, and box shadow
  • Link to HTML rel=""href="URL"
  • img: format image including display, height, width, margin left, and margin right.
  • DRY code: separate similar group elements with ;

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: makes sure you are in the latest version
  • git push: pushes changes
  • git add -A: add to local repository
  • git commit -m: saves changes to local repository
  • git revert SHA: reverts to a commit with a SHA
  • git log: shows all commits in the current branches history

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions tell the computer how to perform a task
  • for loop: perform task on all items in an array by repeating a single code block
  • x++ allows array to itterate
  • conditionals test and expression
